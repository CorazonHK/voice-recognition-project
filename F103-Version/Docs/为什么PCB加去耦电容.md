## 去耦电容详解

> 本文档记录我对去耦电容的理解，包括它是什么、为什么需要、怎么选、怎么放，以及我踩过的坑。

---

### 一、去耦电容是什么？

去耦电容，也叫退耦电容，是**放在芯片电源引脚旁边的电容**。

它的作用用一句话说就是：**芯片需要电流的时候，它能顶上；芯片产生噪声的时候，它能吸收。**

---

### 二、为什么需要去耦电容？

电源不是已经连着了吗？为什么还要额外加电容？


#### 2.1 芯片的电流不是平稳的

芯片内部有几百万个晶体管，时钟一来，一大片同时开关。这个瞬间需要的电流能比平时大好几倍。

用示波器看芯片的电源引脚，会发现电压上有无数尖刺和凹陷。这些都是电流突变造成的。

#### 2.2 电源来不及反应

电源（不管是电池还是稳压芯片）离芯片都有一定距离，中间有走线、有电感。当芯片突然需要大电流时，电源来不及把电送过来，电压就会瞬间掉下去。

电压一掉，芯片就可能复位，或者数据出错。

#### 2.3 去耦电容就是干这个的

在芯片旁边存点电，芯片急着要的时候，先顶上。等这阵过去，电源再把电容充满。

---

### 三、为什么是两个电容？


| 电容  | 容量 | 作用               | 反应速度 |
| ----- | ---- | ------------------ | -------- |
| 10μF  | 大   | 储能，扛突发大电流 | 慢       |
| 0.1μF | 小   | 滤波，滤高频噪声   | 快       |

**10μF 大电容**
- 存的电多
- 芯片突然要大电流的时候，它先放水
- 但对高频信号（几十兆赫兹以上）没啥反应

**0.1μF 小电容**
- 存的电少
- 但反应极快
- 电源线上那些高频噪声，它能直接短路到地

我试过只放 0.1μF：蓝牙一发射就复位（扛不住突发电流）
我试过只放 10μF：不复位，但 ADC 数据跳得像心电图（高频噪声串进去了）

**两个都得要。**

---

### 四、电容怎么选

#### 4.1 容量

- **10μF**：储能用，扛突发电流
- **0.1μF**：滤波用，滤高频噪声

这两个值是经验值，大多数数字芯片都适用。

#### 4.2 封装

| 封装 | 优点     | 缺点             |
| ---- | -------- | ---------------- |
| 0603 | 体积小   | 手焊难，眼神得好 |
| 0805 | 手焊友好 | 体积稍大         |
| 1206 | 好焊     | 占地方           |

新手建议 0805，大小适中，好焊。

#### 4.3 材质

| 材质 | 特点                 | 适用             |
| ---- | -------------------- | ---------------- |
| X7R  | 稳定性好，价格适中   | 10μF 用这个      |
| X5R  | 容量密度高，温漂大点 | 10μF 也可以用    |
| C0G  | 极稳定，但容量做不大 | 0.1μF 用这个最好 |
| Y5V  | 便宜，但性能差       | 别买             |

**我的选择**：
- 10μF：X7R 或 X5R
- 0.1μF：C0G 或 X7R

#### 4.4 极性

- **10μF 有极性**：有一端是正极，上面有色条，或者引脚长的那个是正。焊反会冒烟。
- **0.1μF 无极性**：随便焊。

---

### 五、电容怎么放

这是我第一版板子踩的坑：电容放了，但放得离芯片老远，结果还是不稳。

#### 5.1 位置

**0.1μF 小电容必须紧贴芯片的电源引脚，距离 < 3mm。**

为什么？因为电容离得越远，中间那段走线就越长。走线长了就有寄生电感，电感会阻碍电流瞬间流动。电容存了电，但因为走线太长，芯片急着要的时候电过不去，白存了。

**10μF 大电容可以稍微远一点**，但也不能太远。

#### 5.2 顺序

电源线要先经过电容，再到芯片。

正确：`电源 → 电容 → 芯片`
错误：`电源 → 芯片 → 电容`

如果先到芯片再到电容，电容就成了摆设。

#### 5.3 布线

- 电容的电源脚和地脚，引线要尽量短粗
- 最好直接从电容脚打孔到电源层/地层
- 不要用过孔绕来绕去

#### 5.4 数量

**每个电源引脚都要放一组**。

STM32F103 有好几个 VDD 引脚，每个旁边都要放。只放一个管不了全部。

---

### 六、实测效果

拿我这块 F103 板子试过几种情况：

| 配置                  | 现象                            |
| --------------------- | ------------------------------- |
| 没电容                | 上电都费劲，单片机乱跑          |
| 只放 10μF，远离芯片   | 蓝牙发射时电压掉 0.3V，偶尔复位 |
| 只放 10μF，紧贴芯片   | 不复位了，但 ADC 数据跳 ±15     |
| 只放 0.1μF，紧贴芯片  | 不复位，但 ADC 跳得更厉害，±20  |
| **10μF + 0.1μF 紧贴** | **ADC 跳动 ±2，蓝牙随便发**     |

示波器看电源线，前几种都有毛刺，最后那种干净得像一条直线。

---



### 七、总结

| 项目 | 建议                              |
| ---- | --------------------------------- |
| 容量 | 10μF + 0.1μF 并联                 |
| 封装 | 0805 或 0603                      |
| 材质 | 10μF 用 X7R/X5R，0.1μF 用 C0G/X7R |
| 位置 | 0.1μF 紧贴芯片引脚，<3mm          |
| 顺序 | 电源 → 电容 → 芯片                |
| 数量 | 每个电源引脚一组                  |

去耦电容这东西，有点像给芯片配了个**私人充电宝 + 噪声过滤器**。

- 10μF 是充电宝，芯片突然要大电流的时候顶上
- 0.1μF 是过滤器，把电源线上的毛刺都滤掉

两个少一个，芯片都能工作，但就是不那么稳。时好时坏的那种不稳，最难排查。

现在我画板子，不管什么芯片，不管什么板子，每个电源引脚旁边先放一组 10μF + 0.1μF，已经成习惯了。
